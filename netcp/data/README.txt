Dos programas; NetcpSend.cpp se encarga de enviar el archivo de texto que se encuentra en el mismo directorio "prueba.txt". La dirección ip de envío y el puerto se especifican en las dos primeras lineas del programa. Se compilará con g++ -g NetcpSend.cpp -o sendexe, siendo sendexe el ejecutable que ya se encuetra en la carpeta compresa. El programa NetcpRecieve funciona de manera similar pero se encarga de quedarse a la escucha de mensajes y los muestra por la consola. recexe ese le archivo ejecutable correspondiente a este programa. Actualmente los puertos y direcciones ip especificados permiten enviar através del puerto 51000 de un ordenador a si mismo.
